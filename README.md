# **PRMP Chat**

## A local WLAN chatting software using Sockets

## It includes the following features

> * Private chat.
> * Group chat.
> * Channels.

> * Image Chat.
> * Playing and Recording Audio
> * Audio and Voicenotes chat.

## Unimplementeds

> * Ability to send files (later).
